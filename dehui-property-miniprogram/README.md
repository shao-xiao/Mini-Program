# 德汇创新中心微信小程序

德汇创新中心微信小程序是 DHIC 物业运营平台的移动服务端，面向内部员工、租户用户和关注德汇创新中心的外部客户。

## 项目定位

后台管理系统继续放在 `dehui-property-vscode` 仓库中，小程序仓库只负责微信小程序前端。

后端移动端接口统一规划为：

```text
/api/mobile/**
```

小程序不得直接调用后台运营端敏感接口，应通过移动端接口完成身份隔离、租户数据隔离和移动端数据裁剪。

## 一期 MVP

一期优先实现以下能力：

- 微信登录/开发态登录
- 身份绑定
- 首页
- 我的
- 公告查看
- 租户账单查询
- 会议室预约
- 报修提交
- 招商展示
- 内部员工签到

## 目录结构

```text
config/              环境配置
utils/               请求封装与工具函数
pages/               小程序页面
  home/              首页
  me/                我的
  announcements/     公告
  bills/             账单
  meeting/           会议室预约
  workorders/        报修工单
  investment/        招商中心
  checkin/           内部签到
```

## 开发约定

- 所有网络请求必须走 `utils/request.js`
- API 基础地址从 `config/env.js` 读取
- 移动端接口统一使用 `/mobile/**`
- 不在小程序中写死真实 token
- 不在小程序中越权请求后台运营接口
- 租户数据必须由后端按绑定租户隔离
