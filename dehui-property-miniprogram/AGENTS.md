# 小程序开发守则

本仓库是德汇创新中心物业管理系统的微信小程序前端。

开发前必须先阅读后台仓库中的：

```text
D:\projects\dehui-property-vscode\MINIPROGRAM_PLAN.md
```

## 核心原则

- 小程序以业务办理为主，不照搬后台管理菜单
- 不直接暴露后台运营接口
- 小程序接口统一走 `/api/mobile/**`
- 租户用户只能访问自己绑定租户的数据
- 内部员工权限由后端根据绑定系统用户和角色判断
- 外部客户只能访问公开信息和招商服务
- 所有请求统一走 `utils/request.js`
- 不使用 mock 冒充真实业务闭环

## 当前技术路线

采用原生微信小程序结构，暂不引入 uni-app、Taro 或其它跨端框架。

## 外部依赖

真实上线前需要准备：

- 小程序 AppID
- AppSecret
- 已备案 HTTPS 域名
- 微信支付商户号
- 隐私协议
- 用户服务协议
